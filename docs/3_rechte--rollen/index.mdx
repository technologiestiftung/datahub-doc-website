## So steuert der Data Hub den Zugriff auf Daten

Die Verwaltung von feingranularen Rechten und Rollen ist eine der großen Stärken von urbanen Datenplattformen. Im Data Hub wird die Authentifizierung und Autorisierung für alle Tools zentral von der Komponente Keycloak gesteuert.
Eine ergänzende Governance und ihre Richtlinien sollen zukünftig regeln, wie diese Rollen vergeben werden. So sind Daten sicher, nachvollziehbar und entsprechend der Verantwortlichkeiten nutzbar.

Auf den folgenden Seiten findest du detaillierte Informationen zu den einzelnen Aspekten der Zugriffssteuerung, Rechtevergabe und Data Governance sowie Hinweise dazu, wie diese in der Praxis umgesetzt werden:
